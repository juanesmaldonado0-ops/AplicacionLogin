# AplicacionLogin - Aplicación Android de Bienestar Emocional

Esta aplicación Android está diseñada para ayudar a los usuarios con su bienestar emocional, ofreciendo características como:

- Inicio de sesión seguro
- Diario de emociones
- Sistema de asistente virtual
- Modo de emergencia
- Biblioteca de recursos de bienestar

## Requisitos

- Android Studio Hedgehog | 2023.1.1 o superior
- JDK 17 o superior
- SDK de Android (API 33+)
- Gradle 8.0+

## Configuración del Proyecto

1. Clona el repositorio:
```bash
git clone https://github.com/juanesmaldonado0-ops/AplicacionLogin.git
```

2. Abre el proyecto en Android Studio:
   - File > Open > Selecciona la carpeta del proyecto

3. Sincroniza el proyecto con Gradle:
   - File > Sync Project with Gradle Files

4. Configura un emulador o conecta un dispositivo Android

5. Ejecuta la aplicación:
   - Run > Run 'app'

## Estructura del Proyecto

- `/app/src/main/java/com/example/aplicacionlogin/` - Código fuente Kotlin
- `/app/src/main/res/` - Recursos (layouts, drawables, etc.)
- `/app/src/main/AndroidManifest.xml` - Configuración de la aplicación

## Características Principales

- **Sistema de Login**: Autenticación segura de usuarios
- **Diario Emocional**: Registro y seguimiento de emociones
- **Asistente Virtual**: Apoyo y guía interactiva
- **Modo Emergencia**: Acceso rápido a recursos de ayuda
- **Biblioteca**: Recursos y materiales de bienestar
